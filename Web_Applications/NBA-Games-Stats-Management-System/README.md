# NBA Games Stats Management System (Group 44)

This database project was developed by **Andy Bui** and **Shawn Singharaj** during an 11-week course at Oregon State University. The goal of the project was to learn the fundamentals of designing, creating, and managing relational databases through the process of a full web application.

The system is designed to track **NBA games and player statistics** in a structured database. This allows users to store, view, update, and manage information rather than relying on manual record keeping or spreadsheets.
